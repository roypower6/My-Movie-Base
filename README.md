# **My Movie Base**

> **💡 TMDB Open API를 이용한 영화 검색, 추천 앱**

## **📱 소개**
The Movie DataBase의 Open API를 이용하여 최신 영화, 장르별 영화등 영화 추천부터 영화를 검색하여 정보를 찾아볼 수 있는 앱입니다.
